# 版本更新功能

## 背景

因为版本发布，使用者因为缓存的原因没有及时更新最新页面导致使用的时候出现各种问题

## 案例

该案例以 uniapp vue2 框架为例

## 实现

纯前端实现，编译打包项目时动态生成一个记录版本号的 version.json 文件，
通过轮询，将静态文件 version.json 中的版本号与缓存中的版本号对比是否一致

## 注意事项

在轮询时，调用接口使用的是 uni.request()
需要根据环境决定是否切换 axios
